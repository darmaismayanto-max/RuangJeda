# RuangJeda

> Ruang kecil untuk berhenti sebentar, menaruh rasa, dan melanjutkan hari.

RuangJeda adalah emotional pause engine berbasis React. Aplikasi ini membantu
pengguna menulis perasaan, mengenali mood, menepi sejenak, dan melepaskan beban
tanpa diposisikan sebagai media sosial.

## Fitur

- Beranda kontemplatif dan mood trail tujuh hari
- Confession Wall anonim dengan reaksi empatik
- Mood Journal pribadi
- Mind Space dengan latihan napas dan prompt refleksi
- Release Box dengan animasi pelepasan
- Quote engine berdasarkan mood
- Jeda Suara generatif dengan empat suasana ambient
- Tampilan responsif untuk desktop dan mobile

## Menjalankan secara lokal

```bash
npm install
npm run dev
```

Build produksi:

```bash
npm run build
```

## Safety

RuangJeda bukan layanan konseling profesional dan tidak memberikan diagnosis
atau janji penyembuhan.
